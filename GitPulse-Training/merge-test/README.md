# 融合方法对比实验 (Merge Test)

本目录包含不同文本-时序融合方法的对比实验代码和结果。

## 融合方法

- **门控融合**：使用门控机制融合文本和时序特征
- **注意力融合**：使用注意力机制融合
- **条件初始化**：文本作为初始状态
- **交叉注意力**：时序-文本交叉注意力

## 文件说明

- `fusion_comparison.py` - 融合方法对比实验
- `results/` - 实验结果目录
  - `fusion_results.json` - 融合方法对比结果

## 使用方法

```bash
# 运行融合方法对比实验
python fusion_comparison.py
```

## 结果

融合方法对比结果会展示不同融合策略的效果，帮助选择最优融合方法。
